# Vasicek Credit Risk Model (R)

## Overview
This project implements a Vasicek one-factor credit risk model in R, including simulation, calibration, validation, and portfolio loss analysis.

## Key Features
- Simulated historical default data
- Calibrated PD and asset correlation using MLE
- Performed stability and sensitivity analysis
- Monte Carlo simulation of portfolio loss
- Computed VaR (95%, 99%)

## Skills
- Credit risk modeling
- Maximum Likelihood Estimation
- Monte Carlo simulation
- Model validation
- R programming
